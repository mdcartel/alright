FaintedClient

FaintedClient is a Minecraft client, intended on bending the possibilities of what Minecraft can do, and allowing one to enhance their playing without skill (cheating).

Technologies Used

  • Java
  • Mod Coder Pack
  • Eclipse (not as bad as VS 2022)
  • Discord RPC Client

Features

  • Basic cheats
    • Flight
    • Speed
    • KillAura
    • A lot of GUI elements
    • Disabler (disables some anticheats somehow, including Hypixel years ago)
    • X-ray
    • Auto-clicker
    • ESP
    • NoFall
    • Auto-armor
    • ChestStealer
  • Discord Status Integration
  • Account Manager (incl. Cracked ones)
  • Cool menus
  • Animated wallpaper
  • Really nice ClickGUI (if i say so myself)
