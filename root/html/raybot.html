RayBot

A Discord bot, allowing for interaction of undocumented Epic Games APIS, allowing further enhancement of the enjoyment and possibilities on Fortnite: Save the World

Technologies Used

  • NodeJS
  • DiscordJS
  • Fortnite API (undocumented)

Features

  • Compose MCP Requests (what Save The World uses to communicate with the server)
  • Apply Coupon Codes
  • Duplicate Items...
  • Add/Remove/View Friends
  • View Playtime
  • Give CMD Command to launch Fortnite using any logged-in account (something the Epic Games Launcher cannot do...)
